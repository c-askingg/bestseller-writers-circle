# BestSeller Writers Circle – Programmbegleitung

Dieses Repository enthält einen eigenständigen, responsiven HTML-/CSS-Block für die Landingpage des BestSeller Writers Circle.

## Dateien

- `index.html`: Vorschauseite zur visuellen Prüfung
- `styles.css`: getrennte Styles der Komponente
- `component.html`: reines HTML der Komponente
- `elementor-snippet.html`: vollständig kopierbarer Block für ein Elementor-HTML-Widget

## Einbau in Elementor

1. Ein HTML-Widget an der gewünschten Stelle einfügen.
2. Den vollständigen Inhalt aus `elementor-snippet.html` kopieren.
3. In das HTML-Widget einfügen und aktualisieren.

Die Komponente nutzt keine externen Frameworks, Skripte oder Schriftdateien. Sie übernimmt die Schriftfamilie der bestehenden WordPress-/Elementor-Seite
